使用 Hexo 在 GitHub 部署 Blog

發表於 2020-03-28 更新於 2020-04-06 分類於 Hexo

本來一直不是很想用 Hexo，因為覺得很多人使用，所以有使用另一種 Hugo 來架設 Blog，但是用完一直覺得不合口味，原本打算就將就著用。

剛好看到 Hexo 最多人使用的主題 - Next，感覺還不錯，真香，就來裝裝看吧！
